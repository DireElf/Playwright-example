## Что это

Пример использования Java-библиотеки Playwright для тестирования [учебного веб-приложения](https://qualit.appline.ru/)

## Инструменты

Playwright, JUnit5

## Запуск

Выполнить в командной строке:
```bash
mvn clean test
```
